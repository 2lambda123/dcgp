Documentation¶

The documentation is split into two parts. The C++ documentation and the Python documentation. We try to keep the APIs as similar as possible, but since temmplates are not available in python, we expose an arbitrary number of template instantiations indicating in the class name, after an underscore, the type. For example, the class expression_double corresponds, in Python, to the C++ class expression<double>.

  • C++ Documentation
    • Types of dCGPs
      • dcgp::expression, The standard Cartesian Genetic Program (dCGP)
      • dcgp::expression_weighted, A weighted Cartesian Genetic Program (dCGP-W)
      • dcgp::expression_ann, An artificial neural network Cartesian Genetic Program (dCGP-ANN)
    • Non linearities
      • dcgp::kernel, a function defining the generic dCGP node
      • dcgp::kernel_set: a set of kernels
